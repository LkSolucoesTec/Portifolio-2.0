A plataforma oficial da **LKS Soluções Tecnológicas**. Uma landing page corporativa de altíssima performance arquitetada sob os princípios do design brutalista, código purista (Vanilla JS) e interações cinéticas avançadas.

## ⚙️ Arquitetura e Features

Este ecossistema foi forjado sem o uso de templates, garantindo carregamento instantâneo e controle absoluto sobre o DOM.

* **Design Brutalista / Dark Mode Nativo:** Interface de alto contraste projetada para transmitir autoridade e precisão.
* **Motor Cinético GSAP:** Animações fluidas a 60fps, incluindo *Endless Marquee* (fitas infinitas) de intersecção de dados e *Skew Scroll* (inclinação baseada na velocidade de rolagem).
* **Galeria Magnética:** Cards 3D interativos com *Hover States* complexos e transições de cor (Stealth Mode para Neon).
* **Responsividade Absoluta:** O motor foi calibrado com um "Kill Switch" que adapta perfeitamente a leitura e as animações para dispositivos móveis, empilhando o conteúdo em um fluxo contínuo.
* **Menu Drop-down Tático:** Navegação mobile oculta por um ícone hambúrguer animado, revelando um painel de controle com efeito *glassmorphism* (Vantablack).

## 🛠️ Tecnologias Utilizadas

* **Estrutura:** HTML5 Semântico
* **Estilização:** CSS3 (Flexbox, CSS Grid, Variáveis Nativas, Animações Keyframe, Filtros de Blend Mode)
* **Engenharia Lógica:** JavaScript (Vanilla ES6+)
* **Motor de Animação:** [GSAP (GreenSock Animation Platform)](https://greensock.com/gsap/)
